# Calcaion
Calcaion is a modern multi-purpose calculator developed by AION Technologies. 
Features:
• Standard Calculator
• Scientific Calculator
• Area and Perimeter Calculations
• Length Conversion
• Temperature Conversion
• Angle Conversion
• Time Conversion
• Area Conversion
• Volume Conversion
• Weight Conversion
