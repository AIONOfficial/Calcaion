# 🚀 Calcaion

Calcaion is a modern, high-performance, multi-purpose calculator and unit converter app featuring an intuitive Neo-UI with dynamic theme switching capabilities. 
Designed to deliver both speed and elegance, it bridges the gap between scientific computing and everyday utility tools.

---

## ✨ Features

- **🧮 Computation Suite:**
  - Standard Calculator (Everyday math)
  - Scientific Calculator (Advanced formulas & trigonometry)
- **📐 Geometry Module:**
  - Area and Perimeter calculations
- **🔄 Universal Unit Converter:**
  - Length, Temperature, and Angle conversions
  - Time, Area, Volume, and Weight conversions
- **🎨 Visuals & Systems:**
  - Fluid Dark & Light Neo-Themes
  - **Live Cloud Updates:** Built-in automated online update system.

## 🛠️ Built With

- **Language:** C#
- **Framework:** .NET / Windows Forms
- **Deployment:** Inno Setup Compiler (Standard Windows Installer)
- **Update Engine:** GitHub Cloud-hosted JSON/TXT validation

---

## 📦 Installation & Getting Started

1. Go to the [Releases](https://github.com/AIONOfficial/Calcaion/releases) section on the right side of this repository.
2. Download the latest `Calcaion_Setup_v1.0.0.exe`.
3. Run the installer and follow the setup wizard.
4. Launch **Calcaion** from your Desktop shortcut!
